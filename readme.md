# 基本コーディング環境設定


## 概要

Node.js npm Gulpで各種コンパイルが出来るようにする。

### スタイルシート

SCSS => css

Stylus => css

### HTML
pug => html

ejs => html

## 前提条件

### 下記がインストールされていること

ruby

node.js

compass

## SCSS変換

```
npm run gulp scss
```

## stylus変換

```
npm run gulp stylus
```

## pug変換

```
npm run gulp pug
```

## ejs変換
```
npm run gulp ejs
```


## babel変換

```
npm run gulp babel
```

## 変更を監視しタスクを実行

```
npm run gulp watch
```
